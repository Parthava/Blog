How to start with HTML?

=> First of all you need to know why you are learning the language and how you can apply that in your real life. So the benifit of learning the language is that you can create varoius types of webpages you want. Now the question comes that what are webpages and websites? Simply websites are nothing but the collection of webpages hyperlinked with each other. If you visite any website you can look for the source code (ctrl+u). So start with the basics like the structure of the HTML and different types of tags (source: Getting started with HTML). After going through this topic you will be able to create your first webpage. Start doing experiments, create different webpages, this is the best way to learn HTML. Then move to the next topic that is displaying image in webpage (source: Displaying image). Learn all the attributes of the tag, try them one by one in your webpage. Then move to the main topic of HTML that is Hyperlink (source: Hyperlink). So basically hyperlink is the way to move from one webpage to a another webpage or from a webpage to a document vice versa. These are the main things you need to learn in HTML.
